This project involved creating one page application for a household cleaning company based in Slovakia. THe client had one specific requitement - to make the page navigate like a vertical carousel (one section taking up full viewport space).

> The goal was to keep it as clean as possible.

---

## Visual identity

The key was to create a clean (no pun intended), minimalistic and approachable design, while also keeping a degree of professional feel.

Colors used in this project were kept to minimum, heavily utilising black and white, with navy blue as accent color.

---

## Technical details

The project was designed in Figma and developed using React and Motion library.

---

## Responsivness

The main obstacle with this design was to keep the site responsive across multiple resolutions and orientations. Each section had to fit exactly one viewport height, which proved to be rather difficult for mobile landscape orientation and needed a lot of tweaking.

## Final words

This project proved to be an interesting one. While concept of full screen sections is certainly an interesting and good looking one, in reality it's suitable mostly only for image-heavy sections like hero banners or gallery. While the more text-heavy sections (like FAQ in this particular project) may look good on larger screens, keeping them on one screen height on smaller devices is a challenge.
